Guía de estudio para el JLPT N5: Gramática Parte 1

Siguiendo con la guía para el exámen JLPT N5, ésta semana nos centraremos en la gramática del exámen. Son varios puntos a tratar por lo cual los dividiré en 2 artículos para facilitar la lectura. Si ésta es la primera vez que lees nuestra guía, te recomiendo ver los artículos anteriores sobre los Kanji para el JLPT N5 y las Partículas evaluadas en el exámen JLPT N5.

 

1. です
 

Se usa para indicar el verbo ser. A diferencia del español, sólamente significa ser (no es el verbo ser y estar al mismo tiempo)

  • Ejemplo: わたしはマイクです。 Yo soy Mike.

  • Ejemplo: これはペンです。 Esto es un lapicero.

 

2. ～があります
 

Se usa para expresar el verbo estar o tener. Se usa para referirse a objetos.

  • Ejemplo: コンピューターがあります。 Tengo un computador.

  • Ejemplo: はこの中にプレゼントがありますか。 ¿Hay un regalo adentro de la caja?
     

3. ～がいます


Tiene el mismo uso que ～があります sólo que para referirse a seres vivos.

  • Ejemplo: あそこの木の下に、二人の子どもがいます。 Por allá hay dos niños debajo del árbol.

  • Ejemplo: きょうしつに田中先生がいますか。 ¿El señor Tanaka está en clase?
     

4. ～ませんか


Es una invitación formal. Significa “No te gustaría…?”,“Podemos..?”.

  • Ejemplo: ケーキをたべませんか？ ¿No te gustaría un poco de pastel?

  • Ejemplo: えいがをみませんか？ ¿No le gustaría ( podemos ) ver una película?
     

5. ～ましょう


Es una invitación también. Se usa para decir “Vamos a...”, “Voy a...”.

  • Ejemplo: 食べましょう。 Vamos a comer.

  • Ejemplo: 映画を見ましょう。Vamos a ver una película.
     

6. ～ましょうか


Se usa para hacer sugerencias.

  • Ejemplo: 食べましょうか。 ¿Qué tal si comemos?

  • Ejemplo: 映画を見ましょうか。¿Vemos una película?
     

Los puntos del 7 al 12 se usan seguidos del verbo en su forma TE

 

7. ～てください


Se usa para pedir cosas. Su significado es similar a decir  “Por favor ...”

  • Ejemplo: 日本語ではなしてください。 Por favor hable en japonés.

  • Ejemplo: 彼といってください。 Por favor hable con él.
     

8. ～てもいいです


Se usa para pedir permiso.  Significa “estaría bien…” o “¿puedo…..?”

  • Ejemplo: 料理してもいいですか？。 ¿Puedo cocinar?

  • Ejemplo: 歌ってもいいですか。 ¿Puedo cantar?
     

9. ～てはいけません


Se usa para prohibir, desaprobar o regañar. Significa algo como “No puedes...” o “No sería bueno que...”

  • Ejemplo: サーラーの手書きはいけません。。 La escritura de Sarah no es buena todavía.

  • Ejemplo:いいえ、ざっしをみてはいけません。No, no puedes ver la revista.
     

10. ～ている


Se usa para:

  1. Presente progresivo: Cuando una accion todavia ocurre (En español serían los verbos que terminan en  ～ando, ～endo).

  2. Una acción que se encuentra en cierto estado (que empezó y todavía continúa en el tiempo).

  • Ejemplo: ドアが開いています。 La puerta esta abierta.

  • Ejemplo: 僕は結婚しています。 Yo estoy casado.

  • Ejemplo: 子供は走っています。 Los niños están corriendo.

  • Ejemplo: 私は 読んでいます。 Yo estoy leyendo.
     

11. ないでください


Significa ”Por favor no haga...”  Es usado en los verbos en lugar de la forma TE.

  • Ejemplo: このりんごを食べないでください。 Por favor no coma esta manzana.

  • Ejemplo: その本を読まないでください。 Por favor no lea este libro
     

12. ～にいく


Se usa para decir que vas a “ir a algun lado” o “ir a hacer algo”

  • Ejemplo: 私は泳ぎに行く。 Voy a nadar.

  • Ejemplo: 彼は勉強に行った。 El fue a estudiar.

  • Ejemplo: 彼女は東京に行く。Ella va a Tokyo.
     

13. ～のがすきです


Se usa para decir que te gusta algo.

  • Ejemplo: 私はアニメを見るのが好きです。Me gusta ver anime.

  • Ejemplo: 彼女は歌うのがすきです。A ella le gusta cantar.
     

14. ～のがじょうずです


Se usa para decir que se es hábil en algo.

  • Ejemplo: 彼女は歌うのがじょうずです。 Ella es buena cantando.

  • Ejemplo: あなたは泳ぐのが上手ですか? ¿Eres bueno nadando?
     

15. ～のがへたです


Se usa para decir que eres torpe o malo en algo.

  • Ejemplo: 私は踊るのがへたです。Soy malo bailando.

  • Ejemplo: 彼は勉強するのが下手です。Soy malo para estudiar.

 

Si estás preparandote para el exámen JLPT N5 con ayuda de ésta guía, tienes estas 2 tareas.
 

  1. Debes hacer al menos 5 frases diferentes con cada uno de los puntos de gramática que se tratan en el JLPT N5. 
     
  2. Escribe en los comentarios del artículo una explicación propia ( y usa 2 ejemplos) sobre el punto de gramática que te tomó más tiempo aprender.
     

En caso tengas problemas con algún punto de ésta guía, opina en los comentarios para ayudarte. 皆さん、がんばって下さい。また来週！
